gulpのテンプレート


## 使い方

### npm npm installコマンド
npm install

### gulp起動
インストールされたら、

コマンド「gulp」

でgulpが起動される。

 
 ### v3からv4へアップグレードする方法（ローカル）
１.すでにv3が入っている場合、旧バージョンを削除
コマンド「npm rm gulp --save-dev」

２.最新（v4）をインストール
コマンド「npm i gulp --save-dev」

３.コマンド「gulp -v」を実行し、「Local version 4.x.x」となればOK
